4.What is the deffirence between href attr and src attr?

href (Hypertext REFerence): Specifies the URL of a resource that the user can navigate to or that the browser should link to. It's commonly used with anchor (), link (), and base () tags.
src (Source): Specifies the URL of a resource that should be embedded or displayed directly on the webpage. It's used with elements like images (), scripts ,script,iframe and video.

6.Create a iframe where a single iframe can opens a multiple links in it?

Ans.To create an iframe that can load multiple links, you need to have the links target the same iframe. This can be done using the name attribute on the iframe and the target attribute on the anchor () tags.
Iframe: The iframe is given a name attribute (name="myiframe"), which identifies it.
Links: Each link () uses the target attribute with the value "myiframe", which directs the browser to open the link within the iframe named "myiframe".
